# Savion — Personal Savings Management App

| Project Details | |
| :--- | :--- |

| **Collaboration** | Partnership with [rh45](https://github.com/rh45-one) |

## Overview

Savion is a native Android application designed to streamline personal finance management. Built entirely with **Jetpack Compose**, it leverages modern Android development standards to offer a reactive, user-friendly interface for monitoring savings goals and daily expenses.

## Key Features

Although the project is currently archived, the roadmap included:

* **Financial Tracking:** Transaction logging and income source management.
* **Analysis:** Visual breakdown of savings and expense categorization.
* **Data Management:** Export/Import capabilities for data portability.

## Technical Stack

This project was built using the latest Android standards to ensure scalability and performance:

* **Language:** Kotlin 2.0+
* **UI Framework:** Jetpack Compose (Material Design 3)
* **Concurrency:** Kotlin Coroutines & Flow
* **Architecture:** AndroidX Core KTX
* **Data:** Kotlinx Serialization
